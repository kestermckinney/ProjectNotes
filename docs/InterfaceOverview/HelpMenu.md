# Help Menu

The **Help** menu gives you access to the documentation, keeps Project Notes up to date, and provides a quick way to get assistance when something goes wrong.

## Help

Opens the Project Notes documentation (this guide).

**To open the documentation:**

1. From the **Help** menu, select **Help**.

## What's New

Opens the **What's New** section, which summarizes the features and fixes added in the current release. This is a good place to look right after updating to a new version.

**To see what's new:**

1. From the **Help** menu, select **What's New**.

## Check for Updates

Project Notes can update itself from GitHub. Choosing **Check for Updates…** asks GitHub whether a newer version is available.

- If a newer version is found, Project Notes offers to download and install it. The installer runs automatically and Project Notes relaunches itself when the update completes — there is nothing to install by hand.
- If you are already running the latest version, Project Notes tells you so.

A manual check from this menu always reports its result. Project Notes also checks for updates automatically in the background when **Automatically check for updates** is enabled in [Preferences](Preferences.md); automatic checks stay silent unless an update is available.

**To check for updates manually:**

1. From the **Help** menu, select **Check for Updates…**

## Send Logs to Support

When you run into a problem, the fastest way to get help is to send your log files to support. **Send Logs to Support…** gathers the application's log files (`error.log`, `console.log`, and `syncerrors.log`), packages them into a single zip archive, and opens a new email addressed to support with the archive attached.

You can review the email — and add a description of what went wrong — before sending it.

**To send your logs to support:**

1. From the **Help** menu, select **Send Logs to Support…**
2. Add any details about the problem to the email body.
3. Send the email.

See [Error Log](ErrorLog.md) for a description of what each log file contains.

## About

Displays the **About** dialog, which shows the application name, the version number you are running, and copyright information.

**To view version information:**

1. From the **Help** menu, select **About**.
