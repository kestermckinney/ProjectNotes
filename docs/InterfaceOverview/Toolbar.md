# Toolbar

## Quick Access To Common Functions

The toolbar provides a quick way for you to access some of the more common functions in Project Notes.

![Image](<../img/toolbar-icons.png>)

| **Icon** | **Description** |
| :---: | --- |
| ![Image](<../img/icon-navigate-back.png>) | Navigate back in the page history. |
| ![Image](<../img/icon-navigate-forward.png>) | Navigate forward in the page history. |
| ![Image](<../img/icon-open-item.png>) | Open the highlighted list item. |
| ![Image](<../img/icon-new-item.png>) | Add a new item in the list. |
| ![Image](<../img/icon-copy-item.png>) | Copy the highlighted list item. Fields that must be unique in the list will have a unique value appended in the copied value. |
| ![Image](<../img/icon-delete-item.png>) | Delete the highlighted item. Items that have related project data, can not be deleted until the related project information is deleted or the relationship is changed. |
| ![Image](<../img/icon-plugin-settings.png>) | View the [Plugins Settings](<../StandardPlugins/PluginSettings.md>) window. |
| ![Image](<../img/icon-projects-list.png>) | Navigate to the [Projects List Page](<ProjectListPage.md>). |
| ![Image](<../img/icon-people-list.png>) | Navigate to the [People List Page](<../InterfaceOverview/PeopleListPage.md>). |
| ![Image](<../img/icon-client-list.png>) | Navigate to the [Client List Page](<../InterfaceOverview/ClientListPage.md>). |
| ![Image](<../img/icon-master-item-list.png>) | Navigate to the [Master Item List Page](<MasterItemListPage.md>). |
| ![Image](<../img/icon-filter-tool.png>) | View the [Filter Tool](<FilterTool.md>). |
| ![Image](<../img/icon-search.png>) | Navigate to the [Search Page](<SearchPage.md>). |
| ![Image](<../img/icon-preferences.png>) | View the [Preferences](<../InterfaceOverview/Preferences.md>) window. |
| ![Image](<../img/icon-undo.png>) | Undo recent changes. Only applicable for meeting notes. |
| ![Image](<../img/icon-redo.png>) | Redo recent changes. Only applicable for meeting notes. |
| ![Image](<../img/icon-copy-text.png>) | Copy selected text. Only applicable for meeting notes. |
| ![Image](<../img/icon-cut-text.png>) | Cut selected text. Only applicable for meeting notes. |
| ![Image](<../img/icon-paste-text.png>) | Paste text from the clipboard. Only applicable for meeting notes. |
| ![Image](<../img/icon-text-bold.png>) | Toggle selected text to and from bold. Only applicable for meeting notes. |
| ![Image](<../img/icon-text-italic.png>) | Toggle selected text to and from italic. Only applicable for meeting notes. |
| ![Image](<../img/icon-text-underline.png>) | Toggle selected text to and from underline. Only applicable for meeting notes. |
| ![Image](<../img/icon-align-left.png>) | Left align selected text. Only applicable for meeting notes. |
| ![Image](<../img/icon-align-center.png>) | Center selected text. Only applicable for meeting notes. |
| ![Image](<../img/icon-align-right.png>) | Right align selected text. Only applicable for meeting notes. |
| ![Image](<../img/icon-align-justify.png>) | Justify selected text. Only applicable for meeting notes. |
| ![Image](<../img/icon-indent.png>) | Indent selected text. Only applicable for meeting notes. |
| ![Image](<../img/icon-unindent.png>) | Un-indent selected text. Only applicable for meeting notes. |
| ![Image](<../img/icon-text-color.png>) | View a color selector for the selected text. Only applicable for meeting notes. |
| ![Image](<../img/icon-text-format.png>) | Select the text format for the selected text. Only applicable for meeting notes. |
| ![Image](<../img/icon-font-select.png>) | Select the font for the selected text. Only applicable for meeting notes. |
| ![Image](<../img/icon-font-size.png>) | Select the font size for the selected text. Only applicable for meeting notes. |
