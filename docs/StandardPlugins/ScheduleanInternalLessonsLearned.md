# Scheduling an Internal Lessons Learned Meeting

The **Schedule Internal Lessons Learned** meeting type automatically populates the title of your meeting invitation with the **Project Number**, **Project Name**, and the meeting description. The meeting description is automatically populated with the standard Internal Lessons Learned Meeting Agenda template. For an internal lessons learned meeting, only team members whose company matches the **Managing Company** specified in [Preferences](<../InterfaceOverview/Preferences.md>) are included as attendees. *Note: The Project Manager specified in [Preferences](<../InterfaceOverview/Preferences.md>) is never included in the recipient list.*

Meeting types are configured in **Plugins > Settings > Meeting and Email Types**. See [Outlook Integration](<OutlookIntegration.md>) for email and calendar setup.

**To schedule an internal lessons learned meeting:**

1. Right-click on the project in the **Project List Panel**.
2. Choose **Schedule Meeting > [Internal Lessons Learned meeting type name]**.

When using **Office 365**, the meeting invitation is drafted via the Graph API and works on all platforms. When using **Outlook COM** (Windows only):

3. Make **Outlook** the active application with Alt+Tab or click the **Outlook** icon in the task bar.
4. Edit your meeting time and contents in **Outlook**.
5. Click **Send**.
