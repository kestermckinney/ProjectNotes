# Scheduling a Customer Status Meeting

The **Schedule Customer Status** meeting type automatically populates the title of your meeting invitation with the **Project Number**, **Project Name**, and the meeting description. The meeting description is automatically populated with the standard Customer Status Meeting Agenda template. For a customer status meeting, all team members on the project are included as attendees. *Note: The Project Manager specified in [Preferences](<../InterfaceOverview/Preferences.md>) is never included in the recipient list.*

Meeting types are configured in **Plugins > Settings > Meeting and Email Types**. See [Outlook Integration](<OutlookIntegration.md>) for email and calendar setup.

**To schedule a customer status meeting:**

1. Right-click on the project in the **Project List Panel**.
2. Choose **Schedule Meeting > [Customer Status meeting type name]**.

When using **Office 365**, the meeting invitation is drafted via the Graph API and works on all platforms. When using **Outlook COM** (Windows only):

3. Make **Outlook** the active application with Alt+Tab or click the **Outlook** icon in the task bar.
4. Edit your meeting time and contents in **Outlook**.
5. Click **Send**.
